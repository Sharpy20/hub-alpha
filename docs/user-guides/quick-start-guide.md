# Inpatient Hub - Quick Start Guide

> A practical reference for ward staff to access resources, workflows, and guides quickly.

---

## What is Inpatient Hub?

Inpatient Hub is a centralised resource for ward staff to quickly access crisis contacts, referral workflows, clinical guides, and ward task management. It brings together the resources you need during a shift into one easy-to-use tool.

---

## 5 Things You Can Do Today

### 1. Access Crisis Contacts Instantly

Find emergency numbers and support services in seconds.

- From the **Home** page, use the **Quick Links** carousel
- Click **Bookmarks** in the navigation for the full directory
- Filter by category (Crisis Lines, Mental Health Services, etc.)
- Look for the **FOCUS badge** on links that require Trust network access

### 2. Look Up Referral Workflows

Step-by-step guidance for common referrals.

- Go to **Referrals** in the navigation
- Search or browse by category (Advocacy, Social Care, Physical Health)
- Each workflow includes:
  - Criteria checklist
  - Blank form download
  - Example completed form (WAGOLL)
  - Submission instructions
  - Case note template with **Copy to Clipboard**

### 3. Find Clinical How-To Guides

Quick reference for clinical procedures and assessments.

- Go to **Guides** in the navigation
- Browse available guides (NEWS2, Seizure Management, etc.)
- Each guide provides step-by-step instructions
- Use the **Copy to Clipboard** button to paste templates into notes

### 4. Manage Ward Tasks

Keep track of what needs doing on shift (Medium+ version).

- **Ward Diary**: View all tasks in a weekly calendar format
- **My Tasks**: Kanban-style board showing Pending, In Progress, Completed
- **Claim tasks** to show you're working on them
- **Mark complete** when done
- Switch wards using the **ward dropdown** in the header

### 5. View Patient List

Access patient information and manage discharges (Max+ version).

- Go to **Patients** in the navigation
- Filter by status (Active, Pending Discharge, On Leave, Discharged)
- View patient details, alerts, and named nurse
- **Transfer** patients between wards
- **View Audit Log** for discharged patients

## Getting Started

### Step 1: Log In

1. Click **Log In** in the top-right corner
2. Select your name and role from the dropdown
3. Choose your home ward
4. Click **Enter Hub**

*Note: In the demo version, you can select any user to test different features.*

### Step 2: Accept GDPR Notice

On first visit, you'll see a data protection notice. Click **I Understand** to continue. This appears once per session.

### Step 3: Explore the Home Page

The home page provides quick access to:
- **Quick Links carousel** - Cycle through bookmark categories
- **Quick Actions** - Jump to Bookmarks, Referrals, or Guides
- **Version indicator** - Shows which features are available

## Version Features

Inpatient Hub has different versions with increasing features:

| Version | Use Case |
|---------|----------|
| **Light** | Public resources, personal reference tool |
| **Medium** | Internal SOPs, ward task management |
| **Max** | Patient list, discharge workflows |
| **Max+** | SystemOne integration (coming soon) |

Check your version in the header - it shows as a badge next to your profile.

## Tips for Efficient Use

1. **Bookmark the Hub** - Add it to your browser favourites for quick access
2. **Use keyboard navigation** - Tab through options, Enter to select
3. **Check the ward dropdown** - Make sure you're viewing the right ward
4. **Copy case notes** - Use the clipboard button after completing referrals
5. **Claim tasks early** - Shows colleagues you're handling it

## Quick Reference Card

### Emergency Numbers (Always Available)
- **Samaritans**: 116 123
- **NHS 111**: 111
- **Mental Health Helpline**: 0800 028 0077

### Keyboard Shortcuts
- **Tab**: Move between elements
- **Enter**: Activate buttons/links
- **Escape**: Close modals

### Common Actions
1. Find a phone number → Bookmarks → Search
2. Start a referral → Referrals → Select workflow → Follow steps
3. Look up a procedure → Guides → Select guide
4. Check ward tasks → Ward Diary or My Tasks
5. Transfer a patient → Patients → Select patient → Transfer
